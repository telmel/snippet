Это простейший пример использования react-redux
Здесь все в одном файле.
Как использовать: установить create-react-app redux react-redux
Удалить все файлы из папки src и закинуть туда 1 файл index.js


Теперь можно запускать приложение:
npm start



Особенности:
самопальный какой то редюсер
mapdispatchtoprops не используется и в connect закидываются только mapstatetoprops.
Это значит что в презентационный компонент по умолчанию приходит вместе с пропсами dispatch

Пропсы я десктруктуризую
Затем вручную я диспатчу экшены

В итоге компонент ререндерится при изменении хранилища.
Не инжектим экшн крееторы.  Вместо этого полученный props.dispatch используется для ручного
диспатча экшна
