Этот пример создает простой круг из svg
Используется функция-конструктор которая принимает селектор какого нибудь дива
и несколько опциональных параметров для будущего круга в виде обьекта

Возвращает обьект содержащий DOM обьект с кругом врисованным внутри него

Можно использовать как заготовку для любых разработок связанных с 
вставкой svg - картинок в страницу, не только кругов, но и квадратов
а также сложных фигур.

Можно задавать размеры контейнера и таким образом svg будет заполнять весь 
контейнер, неважно какой большой или маленький он будет
